Using CPU Sim Help

The Help windows for CPU Sim include descriptions of the uses in detail of the menu items, the role and uses of various windows, and a complete description of what hardware, microinstructions, and machine instructions can and cannot be created using CPU Sim. These Help windows can be accessed through the "General CPU Sim Help" menu item in the Help menu or through the Help buttons which appear in most dialog boxes.

The Files tab on the left of the Help window displays a directory tree of all help files. Users can click on the triangle to the left of each folder to see the contents of the folder. Users can click on the files to see the contents of that file.

The Search tab on the left of the Help window allows users to find CPUSim help files on a given topic. As users type in a search string, the help files are searched for the string and the results from the search are displayed in a list of file names below the search box. The list is displayed in a descending order based on the number of occurrences of the search string in the file, which is displayed before each file in the list. Users can click on any file in the list to see the occurrences of the search string in that file. The occurrences are highlighted in yellow. The user can click on the icon next to the search box to restrict the search in several ways.
