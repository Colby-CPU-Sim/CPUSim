A Tour of CPU Sim

To help you get a feel for CPU Sim, we have provided a tutorial. It has three sections. The first section gives the user a tour of CPU Sim and demonstrates how to load into CPU Sim an existing hypothetical architecture called the Wombat1 and then assemble and run a program for that architecture. The second section of this tutorial outlines how an architecture like the Wombat1 is created in CPU Sim. The third section of the tour shows users how to update their machines once they are created.
