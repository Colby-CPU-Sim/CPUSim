Keyboard Shortcuts

There are many keyboard shortcuts for general actions--such as menu item actions--and many shortcuts specifically for editing assembly language programs in text windows. Here is a list of those shortcuts.

Notes:

  • The Command key is used instead of the Control key on Macintosh computers.
  • The Keyboard Shortcuts shown below are the default shortcuts. Nearly all of these can be changed by the user. To change the shortcuts, simply navigate to the "Key Bindings" tab of the Preferences dialog. Click on the keyboard shortcut you wish to change and type in a new shortcut. You needn't press the Ctrl/Cmd key, which is automatically added.

Ctrl-Shift-O      Open machine
Ctrl-B            Save machine
Ctrl-Shift-B      Save machine as
Ctrl-Alt-B        Save Machine in HTML
Ctrl-Shift-P      Print Setup
Ctrl-P            Print
Ctrl-Q            Quit

Ctrl-Z            Undo
Ctrl-Shift-Z      Redo
Ctrl-X            Cut
Ctrl-C            Copy
Ctrl-V            Paste
Ctrl-A            Select all
Ctrl-/            Toggle Comment
Ctrl-F            Find
Ctrl-,            Preferences

Ctrl-M            Open Machine instructions dialog
Ctrl-Shift-M      Open Microinstructions dialog
Ctrl-K            Open Hardware modules dialog
Ctrl-E            Open EQUs dialog

Ctrl-D            Toggle Debug mode
Ctrl-1            Assemble

In addition, all of the standard text navigation shortcuts (Ctrl-left, Ctrl-Right, Ctrl-Delete, etc.) work as expected.
