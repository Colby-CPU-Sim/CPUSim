The Edit Menu

The Edit menu items (see Figure 11) work with any text window when it is appropriate for them to be enabled. The user must first select the tab that the text is contained within to be able to edit that text.


Figure 11. The Edit menu.

Undo/Redo

"Undo" can undo the last change made in the selected text window. "Redo" will redo the last change that was undone. An arbitrary number of undo's and redo's can be done to the changes in any of the text windows.

Cut, Copy, Paste, Delete, and Select All

"Cut", "Copy", "Paste", and "Select All" work in the usual fashion.

Toggle Comment

The "Toggle Comment" will either comment or un-comment a line (or selected set of lines) by adding or removing the machine's comment character at the beginning of the line. The specified comment character can be changed in from the "Punctuation" Tab in the Options dialog.

Find

The "Find" menu item will bring up a dialog that allows the user to search the currently opened text (or IO Console). The user can change their search settings here and "Replace" or "Replace All" instances of a text with another specified text value.

Preferences

"Preferences..." brings up the Preferences dialog. Through this dialog, you can change the font, font size, and font style for the text windows containing assembly code and for the display of the contents of Register and RAM windows. Here, you can also change the key bindings for the keyboard short-cuts for many of the menu items. Finally, the user can also enable automatic saving before assembling a file, show/hide line numbers in the text editors, enable/disable line wrap, and specify whether to clear the console before execution of a program.

Note that on Macintoshes, the "Preferences..." menu item is not in the Edit menu and instead is moved to the Application menu (the leftmost menu, entitled "CPU Sim").
