The Help Menu



The Help menu.

General CPU Sim Help

The "General CPU Sim Help" menu item brings up this online help sytem. The online help is the most up-to-date user manual for CPU Sim.

About CPU Sim

The "About CPU Sim" menu item brings up a dialog box describing the CPU Sim application, including the current version number, information about the authors and contributors, and contact information.

Note that on Macintoshes, the "About CPU Sim..." menu item is not in the Help menu and instead is moved to the Application menu (the leftmost menu, entitled "CPU Sim").
