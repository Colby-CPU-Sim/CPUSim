The Register Arrays Dialog

The Register Arrays Dialog is used to edit the features of each of the registers in register arrays.

The dialog (see Figure 23) has a table in which you can edit the nicknames and initial values of existing registers.


Figure 23. The dialog for editing the register arrays.

The name and initial value cells are edited by typing new text in them, If you try to enter an illegal value in a cell (for example, if you type a non-numeric value in a cell that can only contain a numeric value), an error message will appear when you try to save the changes you made.

The columns in the table can be reordered by dragging the column headers left or right to a new position. The column widths can also be adjusted by dragging the line dividing the column headers. The rows can be resorted by column by clicking in any column header.
